# AI Assistant Suite

A collection of AI-powered tools including a hotkey-driven text assistant and browser extensions for auto-solving quiz questions on Moodle and Classtime.

---

## Components

| Component | Description |
|-----------|-------------|
| **main.py** | Hotkey-driven AI text assistant for processing selected text |
| **server.py** | Local HTTP server for browser extension AI solve features |
| **moodle-extractor/** | Firefox extension for Moodle quiz extraction + AI solve |
| **classtime-extension/** | Browser extension for Classtime.com AI solving |

---

## AI Text Assistant (main.py)

A hotkey-driven desktop assistant that processes selected text using multiple AI providers.

### Supported Providers

| Provider | Description | API Key Required |
|----------|-------------|------------------|
| **OpenRouter** | Cloud models (GPT, Gemini, etc.) | Yes |
| **Ollama** | Local models via Ollama (`localhost:11434`) | No |
| **LM Studio** | Local models via LM Studio (`localhost:1234`) | No |

### Quick Start

**Linux / macOS:**
```bash
chmod +x run.sh
./run.sh
```

**Windows (PowerShell):**
```powershell
.\run.ps1
```

### Hotkeys

| Hotkey | Action |
|--------|--------|
| `CTRL+C` | Copy selected text and process it with AI |
| `ALT+U` | Switch provider and model |
| `ALT+Y` | Quit the program |

### Text Commands

Select text starting with a prefix and press `CTRL+C`:
- **`instruction>`** — Sends the text as an instruction to the AI
- **`grammar>`** — Sends the text for grammar correction

### Configuration

Create a `.env` file:
```
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Place `.txt` files in the `data/` folder for persistent context.

## Browser Extensions

### Moodle Question Extractor

Adds a **📋 Extrahieren** button to Moodle quiz questions (K-Prime, multichoice, gapselect, match). Extracts question text + options with one click, includes AI solve feature.

**Supported types:** `que.kprime`, `que.multichoice`, `que.gapselect`, `que.match`

**Install (Firefox):**
1. Open `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on..."
3. Select `moodle-extractor/manifest.json`

### Classtime AI Solver

Adds **🤖 AI** button to Classtime questions. Auto-detects question type and fills in answers.

**Supported types:** multichoice, singlechoice, categorizer, sorter, cloze, opentext

**Install:**
- Chrome/Edge: Load `classtime-extension/` as unpacked extension
- Firefox: Load `classtime-extension/manifest.json` as temporary add-on

## Local AI Server (server.py)

Required for browser extension AI solve features. Runs on `http://localhost:5923`.

**Start:** `python server.py`

**Configuration:** Uses `.server_config.json` (auto-generated by main.py).

**Default settings:**
- Provider: `openrouter`
- Model: `openai/gpt-oss-120b`
- Port: `5923`

Change defaults in `server.py` or use `ollama`/`lmstudio` for local models.
